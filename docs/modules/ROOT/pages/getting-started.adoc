== 첫 작업 추천

1. xref:catalog/platforms.adoc[플랫폼 등록] — 사내 DB · 데이터 레이크 연결.
2. *메타데이터 동기화* 실행 — 자동으로 데이터셋·스키마 적재.
3. xref:catalog/datasets.adoc[데이터셋] 메뉴에서 적재 확인.
4. xref:catalog/tags-glossary.adoc[태그 · 용어집] 으로 비즈니스 분류 부착.
5. xref:ops/search.adoc[검색] 에서 시맨틱 검색 동작 확인.
