= AI 어시스턴트

tool-use 채팅 어시스턴트다. 카탈로그 검색·스키마·ERD·품질·리니지·표준·SQL 작성을
도구로 호출해 **실데이터를 근거로** 답한다.

== 개요

AI 어시스턴트는 두 가지 모드로 동작한다.

* **외부 에이전트 연동(tool-use)** — `설정 > AI 어시스턴트` 에서 활성화하고 에이전트 URL 을
  지정하면, 외부 독립 에이전트(`agent/ serve`)로 프록시하여 도구를 호출하는 답변을 제공한다.
* **내장 단순 대화(폴백)** — 미설정/비활성 시 LLM 제공자를 통한 단순 대화로 폴백한다.

사용자의 인증 토큰을 그대로 전달하므로, 도구는 **사용자 권한**으로 동작한다.

== 도구(tool)

외부 에이전트 모드에서 어시스턴트가 호출하는 도구의 예:

* 카탈로그/데이터셋 검색
* 스키마 · ER 다이어그램 조회
* 데이터 품질 · 리니지 조회
* 데이터 표준 조회
* SQL 작성 보조

== 활성화

. `설정 > AI 어시스턴트` 탭에서 어시스턴트를 활성화한다.
. 외부 에이전트(`agent/ serve`)의 URL 을 지정한다.
. 외부 에이전트는 로컬 LLM(ollama 등) 또는 OpenAI 호환 엔드포인트에 연결한다.

[NOTE]
====
외부 에이전트(`agent/`)는 자신을 xref:governance/ai-agent-catalog.adoc[AI Agent 카탈로그]
에 등록하고 호출 지표를 보고할 수 있다. LLM 제공자 구성은
xref:ai/metadata-generation.adoc[메타데이터 자동 생성] 과 공유한다.
====

== API

채팅은 SSE 스트리밍으로 제공된다.

[cols="2,3",options="header"]
|===
| 엔드포인트 | 설명

| `POST /api/v1/ai/assistant/chat` | 어시스턴트 채팅(SSE: `text_delta`/`usage`/`done`/`error`)
|===
